30 - 10 - 2019 / 11:17 p.m. The Best Code Editors for developers in 2019
David Torres

For a web page. For a program. For an app Developers and programmers work with programs that allow them to edit the code, work with it and modify it. The options are multiple and varied. We can find them as an independent application, within a programming environment (Integrated Development Environment, IDE) and even in web format.

Choosing one or the other depends on many factors: from the environment for which it is going to be developed (Android, web, Windows ...), the compatibility or not with certain programming languages or the capacities that it has (either innate, or through plugins or extensions).

That is why we will try and compare some of the code or plain text editors that exist today

Visual Studio Code

For web development, no doubt Visual Studio Code of Microsoft is the best editor to work for, especially for productivity improvement.“You can work quickly, efficiently and have extensions and integrations created by users that really make the complete experience of using the editor very powerful. The autocomplete of text, functions, variables is fascinating… ”.

One of the most interesting plugins of VSCode, can be LiveShare. “LiveShare allows you to share your source code in real time in addition to your command console. It is very powerful. You can program as a couple. Simply send a link so that the other person can edit and write, which makes it very useful. ”

IntelliJ IDEA Community Edition

For Java development IntelliJ IDEA Community Edition It stands out for its quite light design, some of the functional ones it offers such as JUNit testing, TestNG, debugging, code autocomplete, code editor for XML and Java, and its integration with some of the most important version control systems, such as Git,

It allows you to import a project and have it all set up in a few steps. In addition, its large set of plugins is also a point in its favor, although it recognizes that "perhaps" this is also the biggest disadvantage that it presents in its CE version: many of these plugins are not available in the free version.

EMACS

An editor must meet a series of key requirements: that it can work in a text terminal, be extensible, cross-platform and have a reasonable life expectancy, which practically forces it to be open source

Not only can you customize even the smallest detail of the editor, but you can trust that there is an extension to do what you want, from using Git with magit to organizing your tasks with org-mode, launching terminals - even running vim within a terminal in Emacs-, read the mail, or quickly program improvised code for some more exotic task (yes, in a Lisp dialect and with a user interface that it is difficult to get used to)

It must be clear that an editor is a tool and not an end. With that as a basis, "it is convenient to choose one and spend time to master it in detail, it is an investment of time that ultimately brings account: in ten years we do not know if Sublime Text will exist, but Emacs and vim will remain there."

Android Studio

It is the IDE that integrates all Android development based on the well-known IntelliJ

Android Studio is a free environment of an incredibly powerful tool

However, we regret that despite being based on IntelliJ, we as Android developers as we are “subject to updates from Google, both on the Android platform and having the latest news from the IntelliJ base, usually more advanced. But "we can't complain about being a free environment of an incredibly powerful tool

Webstorm

Today Webstorm remains the basis for programming both frontend and backend.

WebStorm is a lightweight but powerful IDE, perfectly equipped for complex client-side development and development with server-side Node.js.

However, do not let yourself be carried away by fashions or by what others use without trying to go with you. Simply find the medium that best suits your way of programming, thinking and structuring the code and you feel that it suits you more as a programmer. The important thing is to get good software, a good result, and enjoy along the way. The rest is only the medium, and there are no universal secrets to choose from. For each person it is different, as we are all different
